# scala-cli-gradle-spring-pmd-mockito-cucumber-testng-car-data

## Description
A POC for spring app using testng,
pmd, mockito, and cucumber framework.

## Tech stack
- scala
- gradle
  - spring
  - testng
  - cucumber
  - mockito
  - pmd

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
